# Hướng dẫn thêm ảnh QR Code

## Bước 1: Đặt ảnh QR Code

Đặt ảnh QR code của bạn vào thư mục này với tên: `qr-code-payment.png`

## Bước 2: Đảm bảo ảnh có format

- Tên file: `qr-code-payment.png` (hoặc `.jpg`, `.jpeg`)
- Kích thước khuyến nghị: 300x300px hoặc lớn hơn
- Format: PNG hoặc JPG

## Bước 3: Kiểm tra

Sau khi đặt ảnh, refresh trang `/addfunds` để xem QR code.

## Lưu ý

- Ảnh sẽ được hiển thị khi user nhập số tiền và nhấn "Tạo QR"
- Đảm bảo QR code trong ảnh chứa đúng thông tin tài khoản ngân hàng
